Three web sites that I like the most but are not really well known:




A Soft Murmur
This website lets you create custom ambient sounds to help you relax or focus. I like it because it’s a great way to block out distractions and get work done.



Quora
This is the best place to get answer for a complicated questions. I like it because I can ask questions and get the answer from other user. I also can connect with people who contribute unique insights and quality answers.



Nanoreview
This website helps me to know about gadget. I like it because it’s usefull when I need to compare several devices before decided to buy it. The websites can give a variety of details from camera quality to chipset performance. Therefore, I can get to know which device suits me the best.







Three web sites that related to this course:




W3schools
W3Schools is a popular web development tutorial website that provides free online courses on web development technologies such as HTML, CSS, JavaScript, and PHP.



Notepad++
Notepad++ is a free source code editor and Notepad replacement that supports several languages. However, it is available for Windows only.



Tutorialspoint
Tutorialspoint is another popular website that provides free online courses on various programming languages and technologies.
